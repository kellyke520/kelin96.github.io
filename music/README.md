# 视频解析下载器

此项目仅用于Web开发学习、交流，请勿商用。

## 已知bug

1. 小米手机浏览器加载视频完成后，「左上」和「右上」的圆角失效，显示为直角。（此bug有可能是Android都有的问题，但Android我只测试过小米手机）
2. iOS微信内打开时，翻页到正面后，视频仍会继续播放，几分钟后才停止。**使用Safari浏览器打开即可解决**
